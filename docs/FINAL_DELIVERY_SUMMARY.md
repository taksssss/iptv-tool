## 🚀 使用指南

### 快速启动

```bash
# 1. 安装依赖
cd frontend
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问应用
# 打开浏览器访问 http://localhost:3000
# 使用现有的 IPTV 工具箱管理密码登录

# 4. 使用功能
# - 查看仪表盘
# - 配置管理：编辑 EPG 源、频道别名、定时任务等
```

### 生产构建

```bash
cd frontend
npm run build

# 构建产物输出到 epg/dist/
# 可直接部署使用
```
